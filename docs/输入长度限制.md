# 输入长度限制

本文档记录 Web 管理端与桌面客户端文本输入框的最大字符限制。范围包括 `frontend/src` 与 `desktop/src` 中的文本 `NInput`、原生 `input[type=text]` 与 `textarea`；文件选择、复选框、开关、滑块和数字输入使用文件大小、选项或数值范围限制，不纳入字符长度限制。

## 全局规则

| 类型 | 最大字符数 | 说明 |
| --- | ---: | --- |
| Web 管理端未单独声明的 `NInput` | 4000 | 通过 `frontend/src/App.vue` 的 `NConfigProvider.componentProps.Input.maxlength` 兜底。 |
| 搜索关键字 | 128 | 列表筛选、文档筛选、词库/指令搜索。 |
| 用户名 | 64 | 对齐数据库初始迁移中的 `users.username` 长度。 |
| 密码 | 128 | 后台登录和新增用户表单。 |
| 显示名、设备别名、工作流名、应用名、词库名、模型名 | 128 | 常规名称类字段。 |
| 说话人姓名 | 64 | 声纹注册。 |
| URL、服务地址、回调地址前缀 | 2048 | HTTP(S) 地址类字段。 |
| API Key | 512 | 工作流节点 LLM 配置。 |
| 简短说明、备注 | 512 | 工作流说明、OpenAPI 应用描述、词库说明、声纹备注等。 |
| 词条、敏感词、语气词、替换文本 | 128 | 单个术语或词项。 |
| 误写变体、正则表达式、正则替换 | 1000 | 支持多变体和复杂规则。 |
| 自定义词列表、控制指令候选话术、Meta JSON、回调白名单 | 4000 | 多行配置类字段。 |
| Prompt 模板、节点 JSON 配置、工作流测试文本 | 20000 | 工作流调试与 LLM 提示词。 |
| 桌面会议纪要 Markdown | 50000 | 桌面端会议纪要编辑正文。 |

## 已覆盖入口

| 模块 | 覆盖方式 |
| --- | --- |
| Web 登录、用户管理、OpenAPI、批量转写、会议列表、声纹注册 | 关键字段显式 `maxlength`，其余 `NInput` 走全局兜底。 |
| Web 工作流列表、编辑器、节点默认配置 | 名称、说明、Endpoint、模型、API Key、Prompt、JSON 配置、测试输入均显式限制。 |
| Web 术语库、敏感词库、语气词库、控制指令库 | 词库、词条、规则、候选话术均显式限制。 |
| 桌面连接设置、词库选择、会议列表、会议详情编辑 | 原生文本输入和文本域均显式 `maxlength`。 |

新增输入框时优先复用上表中的类型；确需放宽时，应同步更新本文档并给字段添加显式 `maxlength`。
