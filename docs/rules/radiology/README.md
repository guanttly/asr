# 影像报告书写规则库（rules-catalog）

本目录是影像报告**书写规则**与**ASR 纠错映射**的源文件。每一份 md 文件解析后输出 9 列规则表，对应 `correction_rules` 表的字段。

| 01 | 通用-成像技术与征象 | 单位、乘号、缩写、标点 |

## 修改流程

1. 编辑对应 md 文件，按 9 列表格新增/调整规则。
2. 在项目根目录运行 `make sync-rules-catalog`，自动同步到后端 embed 目录并重新生成 xlsx。
3. 前端「系统管理 → 规则收集 → 影像规则库浏览」可下载最新 Excel；点击「上传 xlsx 导入」即可写入数据库。

## 与术语库的对比

- **术语库（terms）**：词→标准词与别名映射（如「胼胝体」对应错误变体「骈祉体」）。
- **规则库（rules）**：模式→模式替换（如「\d+毫米」→「\d+mm」、「BI-RADS 4」大小写归一）。
- 规则统一挂在 `domain="radiology-rules"`、`name="影像通用规则"` 的字典下。
