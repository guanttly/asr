## 二、系统简介

巨鲨语音助手是一套面向**局域网私有化部署**的端到端语音识别与处理平台，核心能力包括：

- **实时语音识别**：本地分句 + 短句识别，低延迟转写，可即时套用工作流后处理。
- **批量转写**：上传音频文件或提交 URL，异步转写并执行后处理工作流。
- **会议纪要**：上传会议录音，自动转写、说话人分离并由大模型生成结构化纪要（高级版）。
- **声纹库**：注册说话人声纹，用于会议纪要中的实名匹配（高级版）。
- **工作流编排**：可视化编排识别后处理节点（术语纠正、语气词过滤、敏感词过滤、LLM 纠错、会议纪要生成等）。
- **术语 / 语气词 / 敏感词 / 控制指令库**：维护后处理所需的各类词库与规则。
- **OpenAPI 管理**：对外开放语音转写、文本纠错、会议纪要等能力，管理应用凭证与对接文档。
- **桌面客户端**：常驻悬浮球录音、自动注入光标、全局热键、桌面会议纪要与语音控制。

## 三、阅读对象与角色权限

| 用户角色 | 可见模块 |
| --- | --- |
| 管理员 | 数据看板 / 应用 / 工作流 / 术语库 / 系统管理（全部模块） |
| 普通用户 | 应用模块（实时识别 / 批量转写 / 会议纪要） |

> **说明**：除 OpenAPI 管理页（内置“仅管理员可访问”提示）、用户管理页（仅管理员显示“新增用户”按钮）外，前端其余菜单主要校验登录态与产品能力；普通用户能否看到数据或执行操作，最终以后端接口权限返回为准。

## 四、标准版 / 高级版差异

部分能力仅在**高级版**授权下可见可用，未开通时 Web 端隐藏入口、桌面端隐藏对应场景：

| 数据看板 / 实时识别 / 批量转写 | √ | √ |
| 工作流 / 节点管理 / 应用配置 | √ | √ |
| 术语库 / 语气词库 / 敏感词库 / 纠错规则 | √ | √ |
| 用户管理 / OpenAPI 管理 | √ | √ |
| **会议纪要 / 声纹库** | × | √ |
| **控制指令库 / 语音控制** | × | √ |

## 五、通用操作约定

- **登录**：Web 后台默认地址为部署服务器的 HTTPS 入口，使用管理员分配的账号密码登录。
- **安全上下文**：实时录音（Web 与远程访问）需 HTTPS 或 localhost 安全上下文，否则浏览器禁止访问麦克风。
- **音频格式**：上传音频支持 `.wav` / `.mp3`，最终以后端校验为准。
- **业务总线**：页面右上角“业务总线已连接”表示与后端实时消息通道连通，任务状态可自动刷新。
- **版本信息**：左下角显示当前版本号与构建信息，便于反馈问题时核对。
