# 桌面客户端截图占位说明

本目录用于存放**桌面客户端**的截图。手册中桌面客户端章节引用的图片暂为占位，请在客户端运行环境中按下表补充对应截图，文件名需与下方完全一致，放入本目录即可在手册中正常显示。

| 占位文件名 | 建议截图内容 |
| --- | --- |
| `c01-悬浮球.png` | 桌面常驻悬浮球（空闲状态） |
| `c02-悬浮球-录音中.png` | 悬浮球录音中状态（显示当前句段 / 队列） |
| `c03-设置窗-转写记录.png` | 设置窗“转写记录”Tab 列表 |
| `c05-连接设置.png` | 齿轮入口打开的连接设置浮层（服务地址 / 设备别名 / 录入行为） |
| `c06-全局热键设置.png` | 全局热键配置区域 |
| `c07-会议详情-摘要预览.png` | 桌面会议详情，摘要预览模式 |
| `c08-会议详情-摘要编辑.png` | 桌面会议详情，摘要编辑模式 |
| `c09-会议详情-导出PDF.png` | 桌面会议详情，导出 PDF 配置 |
| `c10-识别参数与调试日志.png` | 连接设置中的本地识别参数与调试日志区域 |

> 平台端（Web）截图已由系统真实截取，存放于上一级 [images/](../) 目录，无需手动补充。
