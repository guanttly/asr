# 影像报告易错术语 · 评分规则与说明

> 本目录用于汇总「写影像报告时会遇到」的易错医学术语，按科室分类整理，供 ASR 热词、术语纠错、LLM 后处理校对等用途。

## 1. 收录范围

- 仅收录影像（X 线 / CT / MRI / 超声 / DSA / 核医学 / 乳腺钼靶 / PET-CT 等）报告书写中**真实会出现**的术语。
- 包含：解剖部位、征象描述、技术参数、缩写、量化指标、报告分级（如 BI-RADS / LI-RADS / PI-RADS / TI-RADS / Lung-RADS）。
- 不包含：与影像无关的纯临床用药、护理操作、行政名词。

## 2. 易错维度评分（每项 0–3 分）

| 维度 | 含义 | 0 分 | 1 分 | 2 分 | 3 分 |
| --- | --- | --- | --- | --- | --- |
| **中英文混合度（M）** | 词条中是否夹杂英文字母 / 数字 | 纯中文 | 含 1 个英文字母或缩写 | 中英文 + 数字混合 | 多段缩写 + 参数（如 `T2WI FLAIR`、`PI-RADS v2.1 4 分`） |
| **罕见程度（R）** | 在普通话语料和通用 ASR 训练集中出现频率 | 日常常用 | 医学常用但日常少 | 仅放射 / 病理等专业场景使用 | 罕见综合征 / 解剖变异 / 特定征象 |
| **生僻字（G）** | 词条中是否含生僻字 / 多音字 / 同音易混字 | 无 | 含 1 个生僻或同音易混 | 多个生僻字 | 整词生僻或全为同音陷阱 |

**总分 = M + R + G，范围 0–9。**

## 3. 易错 ASR 等级

| 等级 | 总分范围 | 含义 | 推荐处理 |
| --- | --- | --- | --- |
| **L1（低）** | 0–2 | 常见但偶尔会被 ASR 误识 | 可作普通词库，必要时纠错规则 |
| **L2（中）** | 3–5 | 大概率被误识，需要列入热词 | 加入热词 + 纠错映射 |
| **L3（高）** | 6–9 | 几乎必错，需多变体兜底 | 热词 + 多变体映射 + 上下文 LLM 兜底 |

## 4. 表格字段

每张表使用统一列：

| 字段 | 说明 |
| --- | --- |
| `标准术语` | 报告中应输出的标准写法 |
| `英文/缩写` | 对应英文全称或常用缩写（无则留空） |
| `拼音` | 标准术语的拼音，便于排查同音字 |
| `M/R/G` | 三个维度评分 |
| `等级` | L1 / L2 / L3 |
| `常见 ASR 误识别` | 真实场景中 ASR 易错输出的同音 / 形近变体（用 `、` 分隔） |
| `备注` | 触发条件、上下文说明 |

## 6. 使用建议

1. **热词导入**：将所有 L2/L3 的 `标准术语` 拼接为 ASR 引擎热词。
2. **纠错映射**：把 `常见 ASR 误识别` 作为「误写变体 → 标准术语」喂入术语纠错节点。
3. **校对兜底**：L3 等级触发上下文 LLM 校对（结合报告模板）。
4. **持续迭代**：当线上发现新错误，可在对应科室文件追加行；如出现新分级，更新本表与 README。

## 7. 评分细则的几条裁定

- 「同音字陷阱」 ≥ 2 个时直接 G=2 起步（如「胼胝体／骈祉体」）。
- 含 `PI-RADS / TI-RADS / BI-RADS / GRADE / TIRADS / RECIST` 等带版本号或评分等级的：M≥2。
- 中文医学名 + 英文人名（征象、综合征命名）：M≥1，R≥1（例如「Budd-Chiari 综合征」「Klippel-Feil 综合征」）。
- 计量单位如 `Hounsfield Unit / HU / SUVmax / ADC / FA`：M≥2。

## 8. v2 评测易错规律总结（541 例金标准回归）

> 来源：`docs/tests/ASR评测报告_术语v2.xlsx`（整体 CER 6.28%，剔除离群后）。逐条聚合「替换/删除/插入」错误后发现：**真正拉高错误率的不是生僻术语本身，而是高频报告套语里的单字同音替换**。下列规律按「文件覆盖面（document frequency）」排序，应作为术语易错逻辑的补充——不是逐词收录，而是按「类」处理同音替换。

| 规律类 | 同音替换（金标准字→ASR 误字） | 覆盖文件 | 典型上下文 | 处理建议 |
| --- | --- | --- | --- | --- |
| **报告套语「见」字族** | 见→件 / 建 / 检 / 间 | 187 | 所见、未见异常、未见明显、可见 | 规则：`所件/未件/可件→所见/未见/可见`（句子级 literal） |
| **「余（其余）」误为连词** | 余→与 / 于 | 133 | 余大致正常、余两侧、余未见异常 | 规则：`[与于]大致→余大致`（regex 锚定） |
| **「尚」误为方位「上」** | 尚→上 | 81 | 密度尚可、界尚清、形态尚可 | 规则：`(度\|小\|态\|界\|廓\|限)上(可\|清)→…尚…` |
| **「附/副」误为「腹」** | 附→腹、副→腹 | 71 | 附见：、双侧附件、副脾 | 规则：`腹见：→附见：`、`腹脾→副脾` |
| **「随诊/随访」整词** | 随→史、并→病 | 72 | 临床并随诊、建议随访复查 | 规则：`史诊/史访→随诊/随访`、`临床病随→临床并随` |
| **「压脂」序列名** | 脂→值 / 制 | 28 | T2WI 压脂、脂肪间隙 | 规则：`压值→压脂`；术语行补 `压脂→压值` 变体 |
| **「骶」骨方位** | 骶→底 | 27 | 骶1、骶髂关节、骶管 | 规则：`底髂→骶髂`；编号靠数值归一 |
| **「边缘」征象** | 缘→圆 | 20 | 边缘光整、骨边缘 | 规则：`边圆→边缘` |
| **「纵隔/间隔」解剖** | 隔→膈 | 7 | 纵隔、叶间隔 | 规则：`纵膈→纵隔`、`间膈→间隔` |
| **「终板/皱襞/憩室」专有征象** | 终→中、襞→壁、憩→气 | 26 | 终板炎、黏膜皱襞、肠憩室 | 规则：`中板→终板`、`皱壁→皱襞`、`气室→憩室`；术语行补变体 |
| **「腺」误为「线」** | 腺→线 | 6+ | 甲状腺、前列腺、肾上腺、胰腺 | 规则：`(甲状\|前列\|肾上\|胰\|乳)线→…腺` |
| **「走行/光整/显著」描述词** | 行→形、光→规、著→者/着 | 40 | 血管走行、骨质光整、显著强化 | 规则：`(脉\|管\|支\|束)走形→走行`、`骨质规整→骨质光整`、`显者/显着→显著` |

> **裁定补充**：凡属上述「同音替换类」的术语，`常见 ASR 误识别` 列必须补齐对应误字变体，使其同时进入热词与纠错映射；纯上下文型（如「所见/未见」套语）不单独建术语行，统一交由规则库 `asr-phrase` 冲突组处理，避免误伤正常词。
> 另：金标准里「粘膜」与 ASR「黏膜」属书写规范分歧（`粘→黏` 71 次），**不纳入纠错规则**，保留双写。
