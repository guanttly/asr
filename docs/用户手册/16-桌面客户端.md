# 桌面客户端

> 适用版本：标准版 / 高级版（会议纪要、语音控制为高级版能力）
> 客户端形态：Windows 10/11 使用 **Tauri** 包；Windows 7 SP1 使用 **Electron 22** 兼容包

> **截图说明**：本章节图片均为**占位**，请在客户端运行环境中按 [images/客户端/README.md](images/客户端/README.md) 的命名规范补充截图。

桌面客户端以常驻**悬浮球**提供快速录音，支持本地分句实时转写、自动注入光标位置、全局热键、桌面会议纪要与语音控制。客户端默认通过机器码匿名登录，配置保存在本机。

---

## 1. 客户端启动与窗口结构

![桌面客户端-悬浮球](images/客户端/c01-悬浮球.png)

### 功能特性

- 包含**悬浮录音窗**（悬浮球）与**设置窗**。
- 悬浮球用于快速录音并展示状态（报告 / 会议 / 指令 / 处理中 / 错误）。
- 悬浮球右键可打开设置窗。
- 支持同步全局热键配置；开发模式下可用 `Alt+Shift+D` 打开调试工具。

### 操作步骤

1. 安装并启动客户端，桌面出现常驻悬浮球。
2. **右键悬浮球**打开设置窗，进行连接、历史、会议与参数配置。

### 注意事项

- 桌面端不使用 Web 路由，按窗口标签决定渲染悬浮球或设置页。
- 服务地址、令牌、设备别名、场景、热键与识别参数均**保存在本机**，多窗口间通过本地存储与广播通道同步。

## 2. 连接身份与匿名登录

![连接设置](images/客户端/c05-连接设置.png)

### 功能特性

- 配置服务地址，默认 `http://127.0.0.1:10010`；自动补齐 HTTP 协议并去除末尾斜杠。
- 配置为 HTTPS 且连接失败时，可尝试 HTTP 候选地址。
- 使用**机器码匿名登录**获取 JWT，无需输入账号密码。
- 支持保存设备别名到当前用户资料。

### 操作步骤

1. 打开设置窗，进入连接设置浮层。
2. 填写**服务地址**（指向后台网关或 admin-api）。
3. 客户端自动以机器码匿名登录并加载产品能力与工作流绑定。
4. （可选）填写**设备别名**并保存。

### 注意事项

- 后续请求携带 JWT；遇到 401 时清除旧令牌并重新匿名登录一次。
- 服务地址为空时使用默认地址 `http://127.0.0.1:10010`。

### 异常恢复

| 异常现象 | 处理办法 |
| --- | --- |
| 匿名登录失败 | 设置窗提示失败原因，保留本地配置 |
| 服务健康检查失败 | 提示服务地址或协议可能不正确 |
| 保存别名失败 | 提示失败，不清除当前登录态 |

## 3. 悬浮球录音控制

![悬浮球-录音中](images/客户端/c02-悬浮球-录音中.png)

### 功能特性

- 点击悬浮球开始 / 停止录音。
- 在悬浮球中心区域（约 45% 半径）按住可拖动；短点击仍用于开始 / 停止录音。
- 右键打开设置窗。
- 录音中显示当前句段、队列或错误提示。

### 注意事项

- 音频采集为单声道，目标采样率 16k，约每 200ms 形成一个处理块。
- 未授予麦克风权限时不能开始录音。
- 可配置录音开始后是否自动隐藏悬浮窗。

### 异常恢复

| 异常现象 | 处理办法 |
| --- | --- |
| 麦克风权限拒绝 | 提示未授予麦克风权限，在系统中授权 |
| 未检测到麦克风 | 提示未检测到可用麦克风设备 |
| 麦克风被占用 | 提示麦克风被其他应用占用 |

## 4. 报告模式（实时转写与自动注入）

### 功能特性

- 本地 VAD 判断语音起止并形成短句，短句经 HTTP 接口识别（不维持流式会话）。
- 识别文本可执行实时工作流。
- 开启自动注入时，将最终文本注入当前系统输入焦点。
- 录音结束后保存实时历史任务。

### 操作步骤

1. 确认当前场景为**报告模式**。
2. 点击悬浮球开始录音，对着麦克风说话。
3. 文本经实时工作流处理后自动注入当前光标位置（如已开启自动注入）。
4. 停止录音，满足最小时长且有识别文本时保存为实时历史任务。

### 注意事项

- 实时工作流绑定为空时使用原始识别文本作为最终文本。
- 自动注入默认开启，可在设置中关闭。
- **录音时长小于 1 秒或最终无识别文本时不创建历史记录**。

### 异常恢复

| 异常现象 | 处理办法 |
| --- | --- |
| 短句识别失败 | 记录错误并继续处理后续短句 |
| 实时工作流失败 | 保留原始文本或提示错误，不中断录音 |
| 自动注入失败 | 结果仍保存在历史中，可手动复制 |

## 5. 会议模式（高级版）

### 功能特性

- 会议能力未开通时不展示会议模式。
- 支持从设置窗或热键切换报告 / 会议场景。
- 会议模式录音停止后保存会议任务，使用当前账号绑定的会议工作流。
- 结果在会议列表与会议详情中查看。

### 注意事项

- 会议模式受**产品能力**控制（高级版）。
- 会议模式**不保存为普通实时历史**；满足条件时创建会议任务。
- **录音时长小于 5 秒或最终无识别文本时不创建会议任务**。

### 异常恢复

| 异常现象 | 处理办法 |
| --- | --- |
| 会议能力未开通 | 隐藏会议模式与会议 Tab |
| 会议保存失败 | 提示失败，保留本轮识别状态供处理 |

## 6. 设置窗口与连接设置

![设置窗-转写记录](images/客户端/c03-设置窗-转写记录.png)

### 功能特性

- 设置窗包含**转写记录** Tab；会议能力开通时显示**会议纪要** Tab。
- 齿轮入口打开连接设置浮层，含服务地址、登录身份、设备别名、麦克风权限、录入行为、使用场景、语音控制、全局热键、调试与 VAD 参数。
- 设置保存后同步到悬浮球与其他窗口。

### 注意事项

- 会议 Tab 仅在会议能力开通时显示。
- 设置窗不是完整后台管理页面，不提供 Web 端用户 / 工作流 / 词库的完整管理能力。

## 7. 全局热键

![全局热键设置](images/客户端/c06-全局热键设置.png)

### 默认热键

| 热键 | 动作 |
| --- | --- |
| `Alt+Shift+S` | 显示 / 隐藏设置页 |
| `Alt+Shift+F` | 显示 / 隐藏悬浮球 |
| `Alt+Shift+V` | 开启 / 关闭指令识别 |
| `Ctrl+Shift+Space` | 开启 / 关闭录音 |
| `Alt+Shift+M` | 报告 / 会议模式切换并激活 |

- 另可配置直接激活报告模式、直接激活会议模式的补充热键。
- 支持键盘按键与鼠标侧键 Mouse4 / Mouse5。

### 注意事项

- 同一热键组合不能重复绑定多个动作。
- 会议能力未开通时会议模式热键不生效；语音控制未开通 / 停用时指令识别热键不进入命令模式。
- 设置页支持清空单个热键、恢复默认热键与手动重新同步。
- 指令识别热键在未录音时可自动开启录音；若本次录音由该热键自动开启，则退出命令模式时自动停录。

### 异常恢复

| 异常现象 | 处理办法 |
| --- | --- |
| 热键冲突 | 页面提示冲突动作，不保存重复组合 |
| 后端不支持热键 | 显示注册结果，保留页面配置；不影响鼠标点击录音 |

## 8. 转写历史与快速收录

### 功能特性

- 历史记录从后端实时任务分页接口加载，滚动触底继续加载。
- 优先展示工作流执行后的最终输出，未取到时回退原始识别结果。
- 支持复制最终文本、注入当前焦点、将选中文本收录到术语库、加入敏感词库。
- 支持删除单条历史与清空历史。

### 操作步骤

1. 在设置窗**转写记录** Tab 查看历史。
2. 对某条记录：**复制** / **注入** / **收录术语** / **加入敏感词**。
3. 收录术语时可选择已有词库或新建词库。
4. 删除单条或清空历史前需**二次确认**。

### 注意事项

- 清空历史只清理当前账号下可删除的实时任务，进行中的任务可能被跳过。
- 注入依赖操作系统输入模拟能力。
- 桌面端快速收录仅提供便捷写入入口，非完整词库管理。

### 异常恢复

| 异常现象 | 处理办法 |
| --- | --- |
| 历史加载失败 | 提示加载失败并允许刷新 |
| 词库收录失败 | 提示失败原因，保留弹窗输入 |
| 注入失败 | 提示失败，可改用复制 |

## 9. 桌面会议列表与详情（高级版）

![会议详情-摘要预览](images/客户端/c07-会议详情-摘要预览.png)

### 功能特性

- 会议列表支持搜索、刷新、分页；处理中会议每约 6 秒轮询刷新。
- 会议详情在摘要页支持**预览 / 编辑 / 导出**三种模式，并提供逐字稿页。
- 摘要编辑支持标题、会议主题、会议时间、会议地点、主持人、记录人与 Markdown 正文。
- 支持保存摘要编辑、重新生成摘要。
- 导出模式支持配置标题、会议信息、逐字稿、字号与强调色后导出 **PDF**。

### 操作步骤

1. 在设置窗**会议纪要** Tab 查看会议列表。
2. 打开会议详情，在摘要页切换模式：
   - **预览**：按 Markdown 渲染当前摘要；
   - **编辑**：修改标题与正文等字段后**保存**（以服务端返回内容为准）；

     ![会议详情-摘要编辑](images/客户端/c08-会议详情-摘要编辑.png)
   - **导出**：配置导出选项后导出 PDF。

     ![会议详情-导出PDF](images/客户端/c09-会议详情-导出PDF.png)
3. 需要时**重新生成摘要**；删除会议前需二次确认。

### 注意事项

- 会议 Tab 仅在会议能力开通时显示。
- 会议详情处理中时每约 4 秒轮询刷新。
- 导出仅支持 **PDF**（依赖 html2pdf 与 Tauri 原生保存能力），不提供单独的 Markdown 文件导出；逐字稿是否附加到 PDF 由导出选项决定。

### 异常恢复

| 异常现象 | 处理办法 |
| --- | --- |
| 会议列表加载失败 | 提示失败并允许刷新 |
| 摘要保存失败 | 保留编辑内容并提示失败原因 |
| PDF 导出失败 | 提示导出失败，不影响会议数据 |
| 会议删除失败 | 保留列表数据并提示失败原因 |

## 10. 语音控制（高级版）

### 功能特性

- 普通语音片段先执行唤醒识别，命中后进入命令模式。
- 命令模式下识别文本只用于指令分类，不写入历史或注入。
- 支持切换报告模式；会议能力开通时支持切换会议模式。
- 支持通过热键直接进入 / 退出指令模式。

### 注意事项

- 指令识别调用当前账号绑定的**语音控制工作流**（见[应用配置](08-应用配置.md)）。
- 命令等待时间来自应用配置，桌面端最低按 2 秒处理。
- 连续 3 次分类失败后退出命令模式，避免持续吞掉语音。
- 切换场景后刷新工作流绑定，下一句使用最新绑定。

### 异常恢复

| 异常现象 | 处理办法 |
| --- | --- |
| 未绑定语音控制工作流 | 不进入有效指令识别，提示或记录失败 |
| 指令分类超时 | 退出命令模式并恢复原场景 |
| 未开通语音控制能力 | 隐藏语音控制配置与相关热键动作 |

## 11. 识别参数与调试日志

![识别参数与调试日志](images/客户端/c10-识别参数与调试日志.png)

### 功能特性

- 配置是否保留标点。
- 配置最小触发阈值、底噪倍数、静音块数、有效语音块数、单块峰值倍数。
- 请求麦克风权限。
- 开启 / 关闭调试日志，刷新最近日志、复制日志、打开控制台。

### 默认识别参数

| 参数 | 默认值 |
| --- | --- |
| 保留标点 | 关闭 |
| 最小触发阈值 | 0.018 |
| 底噪倍数 | 2.8 |
| 静音块数 | 4 |
| 有效语音块数 | 2 |
| 单块峰值倍数 | 1.45 |

### 注意事项

- 识别参数保存到**本地设置**，不写入服务端；对后续录音生效，不回改已保存历史。
- 调试日志用于本机问题诊断，不作为业务数据展示。
- Web 实时识别与桌面实时识别使用相同思路的本地 VAD 参数，但分别存储在各自客户端环境中。

### 异常恢复

| 异常现象 | 处理办法 |
| --- | --- |
| 参数不合法 | 使用默认值或限制范围内的值兜底 |
| 麦克风权限请求失败 | 显示失败原因并保留未授权状态 |
