## 2. 分布速览

### 2.1 按等级（占比）

| 等级 | 数量 | 占比 | 含义 |
| --- | ---: | ---: | --- |
| **L1** 低 | 338 | 31.5% | 偶尔会错，常规词库覆盖 |
| **L2** 中 | 464 | 43.2% | 大概率被误识，必须进热词 + 误写映射 |
| **L3** 高 | 271 | 25.3% | 几乎必错，需多变体 + LLM 兜底 |

### 2.2 L3 集中度（高错风险最大的科室）

> L3 占比 = L3 / 该科室总数

1. **神经-颅脑脊髓**：L3 = 41 / 136（30%）— 胼胝体、穹窿、髓样、岩部等生僻字密集
2. **腹部-肝胆胰脾消化道**：L3 = 33 / 111（30%）— 大量人名命名（Vater、Budd-Chiari、Klatskin 等）
3. **肌骨-骨关节脊柱**：L3 = 32 / 113（28%）— 骨骺、干骺端、髌、跖、Salter-Harris 等
4. **肿瘤分期与报告标准**：L3 = 30 / 62（48%）— RADS 系列 + RECIST/PERCIST 等评分体系
5. **通用-成像技术与征象**：L3 = 26 / 111（23%）— FLAIR、SWI、钆对比剂、SUVmax 等

### 2.3 误识模式分类

| 模式 | 典型例子 | 应对 |
| --- | --- | --- |
| **同音字** | 灶→照、栓→拴、晕→运、髓→随、囊→能 | 误写变体映射 |
| **生僻字** | 钆、骺、髌、痂、贲、骶、颞、矽 | 热词 + 拼音强提示 |
| **多音字** | 壳（qiào）、奇（qí）、塞（sè） | 上下文 LLM 校对 |
| **中英混合** | T1WI、MRCP、PI-RADS、SUVmax、Y-90 | 必须进热词；保留原大小写 |
| **外国人名征象** | Klatskin / Budd-Chiari / Schmorl / Bochdalek / Mirizzi | 多变体 + 译音映射 |
| **缩写拆读** | ASD→艾思低；DSA→低 S A | 上下文 LLM 兜底 |
| **数字+字母** | C1、L5、S1、T12、BI-RADS 4A | 模板正则修正 |

## 3. 如何接入项目

### 3.1 热词列表（ASR 引擎）

将所有 **L2 + L3** 的 `标准术语` 拼接为热词集（~735 条）。建议格式：

```text
# /backend/.../hotwords/radiology.txt
胼胝体
颞叶
枕叶
FLAIR
T2WI
LI-RADS
PI-RADS v2.1
...
```

> 与 [docs/产品规格书.md §3.9 术语库管理](../产品规格书.md) 中的「标准术语 + 误写变体」字段对齐，可作为初始库导入。

### 3.2 纠错映射（术语纠错节点）

每行的 `常见 ASR 误识别` → `标准术语`，喂入工作流术语纠错节点（见 [docs/产品规格书.md §3.13 术语纠错规则](../产品规格书.md)）。

```yaml
# 示例
- standard: 胼胝体
  variants: [骈祉体, 便支体, 片织体]
- standard: 钆对比剂
  variants: [嘎对比剂, 咖对比剂, 加对比剂]
- standard: PI-RADS
  variants: [屁瑞兹, pi rads, p i 瑞兹]
```

### 3.3 LLM 后处理（仅 L3）

对包含 L3 命中或邻近上下文的语段，启用 LLM 校对（参考 [概要设计.md] 中 LLM 纠错节点）。提示词建议：

> 这段话来自影像科报告，疑似 ASR 误识。请在不改变临床含义的前提下，按本机构标准术语进行最小修正，输出修正后文本与变更摘要。

### 3.4 模板正则（结构化短语）

对 RADS / TNM / 解剖编号等结构化片段，使用正则收尾：

```text
# BI-RADS x 类
(BI[\-\s]?RADS)\s*([0-6][A-C]?)\s*类?
# 椎体编号
[CTL]([1-9]|1[0-2])\b
# Couinaud
[Ss]([1-9])\s*段
```

## 4. 后续迭代建议

1. **真实语料挖掘**：定期从线上术语纠错日志统计高频「误写 → 标准」对，反哺到本表。
2. **本院别名收录**：每个医院都有内部俚语（如「LDLD = 双下肢」），可在每张表新增「本院别名」列。
3. **多模态校验**：模板触发时，结合检查部位字段（DICOM `BodyPartExamined`）二次过滤，避免误纠（例如脑科室不该出现「肝段」）。
4. **版本号管理**：当 RADS / TNM 推出新版（如 LI-RADS v2023、AJCC 第 9 版），保留旧版条目并新增版本字段。
5. **自动评分校准**：建议未来跑一次线上日志，把 ASR 实际错误率反映到 M/R/G 三维分数中，使评分动态化。
